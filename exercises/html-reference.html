HTML Tag

The DOCTYPE html is not a tag.  The purpose of it is to instruct the program on what language the page will be written in.  The html tag after that has the same purpose of setting up the page to be written in html.

Head Tag

The purpose of the head tag is to exclude the "written" text from the actual page.

Title Tag

The title tag is included in the head tags where the title is not visible/written on the actual page. The purpose of it is just to name the page itself.

Body

Body tags are used in order to break up text that you want to display on your website.  These are essential to be able to actually have written text on your website

Paragraph

"p" or paragrah tags are used in order to write smaller text below the heading text.  This is very important for having basic text on your website.

Breaks

Break tags "br" are used when you want to separate different text or images so that they are not side by side. Several break tags were used in this website to separate the text.

Lines

Line tags "hr" have the same function as break tags but are much more pronounced since they display an actual line that you can see.

Comments

Comments can be used in order to leave notes about your website without displaying text. It can also be used to temporarily remove code without completely losing it.

Links

Link tags can be used in order to reference external websites/videos or to make a compound website where there are multiple pages that you need to link up.

DIV

DIV block tags are used in order to give differen text and images attributes that can be controlled.  These are useful when you don't want the entire page having the same attributes and only for specific parts or the site.
