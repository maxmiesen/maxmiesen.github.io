HTML

Description

Sets the coding language for the page as "html"

Syntax

start tag: <html>
content: html elements
end tag: </html>

Example

<html> 

  <head> 

    <title>Title Name</title> 

  <head> 
 
<html>

Head

Description

Container for information that is not displayed on the page itself (things such as the title, styles, etc.)

Syntax

start tag: <head>
content: head elements (title and styles)
end tag: </head>

Example

<html> 

  <head> 

    <title>Title Name</title> 

  <head> 
 
<html>

Title

Description

The title tag is displayed at the top of the web browser and it briefly describes your site by naming it

Syntax

start tag: <title>
content: Title Name
end tag: </title>

Example

<title>CNN</title>

Syntax

content: Body Elements
end tag: </body>

Example

  <p> Insert paragraph here </p>

Headings

Description

Heading tags are defined text styles set to a default. You can change these and "style" them using CSS in order to personalize your website

Syntax

start tag: <h1>
content: Heading text
end tag: </h1>

Example

<h1> Chapter Tweleve: Homework Sucks </h1>

Paragraph

Description

The larger amounts of text in paragraph format are normally used with this tag.

Syntax

start tag: <p>
content: Paragraph text
end tag: </p>

Example

<body> 

  <p> This is an example of a paragraph where I'm writing nonesense but showing how there is usually more writing here than there is in a heading tag </p> 

</body>

Breaks

Description

Breaks make your website easier to read because without them, you would have to scroll very far to the right to read everything.

Syntax

tag: <br>

Example

<p> Paragraph here <br> 

Paragraph here </p>

Lines

Description

Lines can divide up the different text and images on your website and organizes everything.

Syntax

tag: <hr>

Example

<p> Paragraph here <hr> 

Paragraph here </p>

Comments

Description

Comments can organize your code, you can leave reminders that explain confusing code, and you can also trouble shoot different parts of your code to see what isn't working.

Syntax

start tag: <!--
content: Comment text
end tag: -->

Example

<!-- This #FFDF00 RGB code is gold -->

Links

Description

You can include links to reference other pages made by different people or linking up your website with different pages

Syntax

start tag: <a href=>
content: link url inside a href and text outside
end tag: </a>

Example

<a href="https://youtube.com">YouTube</a>

Images

Description

end tag: </div>
